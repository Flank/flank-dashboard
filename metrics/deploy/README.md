# Metrics deploy CLI

A simple command-line application build with Dart to deploy metrics app to GCP.

# Build

```
make build
```

# Usage

Get supported commands.

```
build/metrics
```


Check env and dependencies.

```
build/metrics doctor
```

Deploy 

```
build/metrics deploy
```
